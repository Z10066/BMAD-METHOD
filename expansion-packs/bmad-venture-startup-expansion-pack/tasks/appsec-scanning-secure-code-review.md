# AppSec Scanning & Secure Code Review

## Purpose

落地SAST/DAST/SCA与代码审查要点。

## Inputs

- 代码仓/流水线/配置
- templates/secure-code-review-checklist-tmpl.yaml
- checklists/vulnerability-triage-and-sla.md

## Outputs

- 扫描策略与代码审查清单。

## Steps

1. 选择工具与覆盖率/阈值
2. 误报管理与基线更新
3. 拉取请求安全检查与必需项
4. 高危缺陷优先级与SLA
5. 季度报告与培训
