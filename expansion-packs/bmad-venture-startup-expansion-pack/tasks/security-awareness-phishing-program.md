# Security Awareness & Phishing Program

## Purpose

员工安全意识/钓鱼模拟与培训。

## Inputs

- 培训平台/题库/邮件模板
- templates/awareness-and-phishing-plan-tmpl.yaml
- checklists/phishing-sim-and-awareness.md
- data/phishing-red-flags.md

## Outputs

- 年度培训计划与钓鱼报告。

## Steps

1. 入职/年度/专项培训节奏
2. 钓鱼模拟与得分/复训
3. TOP错误与防范提示
4. 合规记录与通过率
5. 改进与奖励机制
